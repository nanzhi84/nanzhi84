## 👋 About me

- 🎯 &nbsp;AI undergrad in Beijing, building at the intersection of **agents, digital humans, and LLM post-training**.
- 🎬 &nbsp;Building **树影 · Cutagent** — a *Case-first* digital-human short-video production system with a real **TTS → ASR → lip-sync → render** pipeline (Temporal runtime, strongly-typed contracts).
- 🛠️ &nbsp;Crafting **agentic dev workflows** — Claude Code × Codex collaboration skills, plus [`harness-sync`](https://github.com/nanzhi84/harness-sync) to carry an AI-coding setup *losslessly* across machines.
- 🔬 &nbsp;Researching **reward models & LLM post-training** — the `RMGAP` line of work.
- 🌐 &nbsp;Always curious about **AI + social**.
- ⚡ &nbsp;Most days you'll find me commit-bombing one of these repos. 👇
